# Production Checklist

- [ ] Production secrets configured.
- [ ] Migrations deployed.
- [ ] Seed ran successfully.
- [ ] Health readiness checked.
- [ ] Backup job verified.
- [ ] Restore test completed in staging.
- [ ] CORS and CSRF origins locked down.
- [ ] API and web images built as non-root.
- [ ] Desktop installer signed and tested.
- [ ] Sync conflict policy reviewed with operations.
- [ ] Monitoring alerts connected to health and metrics.
